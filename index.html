LitMarks.ai

Export your Amazon Kindle highlights.

Your highlights just get buried—you never go back. Surface them. Export to JSON and turn them into useful things you'll actually check.

Install from Chrome Web Store

Features

Select books

Choose which books to export. Use Select all or Clear for bulk actions.

JSON export

Structured output with titles, authors, highlights, colors, and locations.

One-click

Works directly on the Kindle notebook page. No extra steps.

How it works

  1. 1

    Open read.amazon.com/notebook and log in.

  2. 2

    Click the LitMarks icon in the Chrome toolbar.

  3. 3

    Select the books you want and click Export selected.

  4. 4

    Download the JSON file when done.

Output format

Download the JSON file and use it in any integration you want—Notion, Obsidian, your own app, or anywhere else.

{
  "extractedAt": "2025-02-06T...",
  "books": [
    {
      "asin": "B00LMGLXTS",
      "title": "Hooked: How to Build Habit-Forming Products",
      "author": "Nir Eyal and Ryan Hoover",
      "highlights": [
        {
          "text": "...",
          "color": "yellow",
          "location": "75"
        }
      ]
    }
  ]
}
